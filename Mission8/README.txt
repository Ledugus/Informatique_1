A l'exécution de "mission8.py", le programme :
 - lit un fichier "music-db.txt",
 - le convertit en album,
 - imprime les albums. 

Les classes Duree, Chanson et Album sont définies dans le fichier "mission8.py".

Le fichier "test.py" teste les méthodes des classes Duree, Chanson et Album.
